Cara jalanin:

1) MySQL:
   - Jalankan server MySQL.
   - Eksekusi file 'fkik-backend/README_DB.sql' untuk membuat DB & tabel.

2) Backend:
   cd fkik-backend
   npm install
   node server.js

3) Frontend (Vite React):
   cd ../dashboard-fkik
   npm install
   npm run dev
